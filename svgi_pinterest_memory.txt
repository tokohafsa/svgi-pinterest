## AKUN
- Pinterest: svg_i (niche: logo animation)
- IG source konten: repost dari kreator logo animation di Instagram

## TUJUAN APP
Automate pipeline dari Instagram / Direct URL → Dropbox → Pinterest CSV bulk upload

## STACK
- Framework: Streamlit (deployed ke Streamlit Cloud)
- AI/Model: Groq API (llama-3.3-70b-versatile)
- Video fetch IG: yt-dlp
- Video fetch Direct: requests (download langsung)
- Frame extraction: imageio-ffmpeg (bundle binary)
- Storage: Dropbox API (App Folder: /Apps/SVGI Pinterest/)
- Output: CSV + video tersimpan ke folder session di Dropbox

## PIPELINE FLOW — TAB 1 (Instagram)
1. New Session → buat folder Dropbox: /2026-07-23_1430
2. Paste URL Instagram reel/post (support carousel dengan ?img_index=N)
3. Fetch metadata (yt-dlp) + download video
4. Upload video ke Dropbox
5. Extract 8 frame thumbnail (ffmpeg)
6. Preview video + thumbnail grid (klik untuk pilih, border merah tanda terpilih)
7. Raw caption ditampilkan (read-only)
8. Credits: field Poster @ + @Mentions dari caption (auto-detect, editable)
   Preview credit line realtime di bawah field
9. Input pin name + pilih Board + pilih Sector
10. Generate → review/edit Title, Description, Keywords
11. Regenerate jika perlu
12. Add to Queue
13. Loop ke step 2 untuk URL baru
14. Generate CSV Preview → Edit CSV → Upload ke Dropbox (3 step terpisah)

## PIPELINE FLOW — TAB 2 (Direct URL / GIF)
1. Paste direct URL (.mp4 / .mov / .gif)
2. Download file + extract frames thumbnail
3. Preview video/GIF + thumbnail grid
4. Input manual: Nama pin, Credit animator (plain text), Link ke konten asli (dinamis per pin)
5. Pilih Board + Sector
6. Generate → review/edit Title, Description, Keywords
7. Add to Queue
8. Generate CSV Preview → Edit CSV → Upload ke Dropbox (3 step terpisah)
   Nama file CSV: NON-IG_bulk_logo_{session_folder}.csv

## CAROUSEL FIX (instagram.py)
- _parse_img_index(url): parse ?img_index=N dari URL
- _strip_img_index(url): buang parameter sebelum kirim ke yt-dlp
- yt-dlp fetch seluruh carousel dengan ignoreerrors=True
- _pick_entry(info, img_index): filter entries yang video (bukan foto),
  lalu ambil entry ke-N sesuai img_index (1-based)
- Fallback: kalau tidak bisa filter, pakai index mentah

## CREDIT LINE FORMAT
- Ada @mention di caption:
  "Credit: @mention1 @mention2 | from @poster on Instagram"
- Tidak ada @mention:
  "Credit: @poster on Instagram"
- Tab Direct (NON-IG):
  "Credit: {nama animator yang diinput}" — plain text, tidak ada @poster

## FORMAT TITLE
{Nama} {suffix board}
Suffix per board:
- Logo Animations             → "Logo Animation"
- E Sports Gaming ...         → "Logo Animation"
- Famous Brand ...            → "Logo Animation"
- Brand Identity in Motion    → "Animated Brand Identity"

Contoh board biasa: "Vortex Logo Animation. [CTA]"
Contoh Brand Identity: "Nike Animated Brand Identity. [CTA]"

## FORMAT DESCRIPTION
Line 1: {credit_line}
Line 2-3: seo_body (2 kalimat SEO, max 200 chars)
Line 4: {cta_desc}

Catatan:
- Title TIDAK diulang di baris pertama description
- Board "Brand Identity in Motion": seo_body harus bicara tentang brand system
  dan visual identity, bukan logo reveal atau logo intro
- Jangan sebut 3D, three-dimensional

## SCHEDULER (scheduler.py)
Parameter: end_date, count, start_date (opsional), shuffle
- start_date = hari ini atau kosong → range mulai dari now + 2 jam
- start_date = besok atau lebih     → range mulai dari 02:00 pagi hari itu
- range_end = end_date 23:59:59
- Distribusi merata, interval rounded ke menit terdekat

## EXPORT CSV FLOW (3 STEP)
Step 1: Generate CSV Preview
  - Pilih enable/disable scheduling
  - Pilih start_date + end_date
  - Klik "Generate CSV Preview" → isi tabel editor, TIDAK langsung upload
Step 2: Edit CSV
  - st.data_editor — semua kolom editable (Title, Description, Board, Keywords, dll)
  - Board pakai SelectboxColumn dengan opsi BOARDS
Step 3: Upload ke Dropbox
  - Klik "Upload CSV ke Dropbox" → baru simpan ke Dropbox
  - Tampilkan path + tombol "Upload to Pinterest"
