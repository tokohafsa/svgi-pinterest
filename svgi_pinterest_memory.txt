## PIPELINE FLOW (per pin)
1. Paste URL Instagram reel/post
2. Fetch metadata (yt-dlp) + download video + upload ke Dropbox otomatis
3. Preview video (dari Dropbox link) — 50% width
4. Set thumbnail timestamp (e.g. 0:04)
5. Raw caption ditampilkan
6. Credits auto-detect dari caption (editable): client, animator, logo_maker
7. Input pin name opsional + pilih Board + pilih Sektor (opsional, untuk SEO)
8. Generate → review/edit Title, Description, Keywords
9. Regenerate jika perlu
10. Add to Queue
11. Loop untuk URL baru
12. Export CSV (toggle schedule ON/OFF)

## CREDIT FORMAT RULES
- animator: SELALU = uploader_id (akun yang posting di IG)
- client: @mention yang disebut sebagai pemilik logo / "for @", "with @"
- logo_maker: @mention yang disebut sebagai logo designer ("logo by @", "design by @")
- Jika tidak ada client: "Credit: @animator on Instagram"
- Jika ada client: "for @client | Credit: @animator on Instagram"
- Jika ada logo_maker: tambahkan "| Logo by @logo_maker"

## UPLOADER_ID DETECTION (instagram.py)
Priority order:
1. uploader_url field → parse username dari URL instagram.com/username/
2. uploader_id field → hanya jika bukan numerik dan tidak ada spasi
3. channel field → fallback
4. channel_url → parse username
5. @mention pertama di caption → last resort
