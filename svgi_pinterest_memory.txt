## PIPELINE FLOW (per session)
0. Klik New Session → buat folder Dropbox: /2026-07-11_1430
1. Paste URL Instagram reel/post
2. Fetch metadata (yt-dlp) + download video
3. Upload video ke folder session Dropbox
4. Extract 8 frame thumbnail (ffmpeg) dari file lokal sebelum dihapus
5. Preview video (dari Dropbox link) — max height 300px, 50% width
   + fallback link button untuk Safari/iPadOS
6. Thumbnail grid: 8 frame ditampilkan, klik untuk pilih → border merah tanda terpilih
   Fallback ke radio button jika ffmpeg gagal
7. Raw caption ditampilkan (read-only)
8. Credits auto-detect dari caption (editable): client, animator, logo_maker
9. Input pin name opsional + pilih Board + pilih Sektor (opsional, untuk SEO)
10. Generate → review/edit Title, Description, Keywords
11. Regenerate jika perlu
12. Add to Queue
13. Loop ke step 1 untuk URL baru
14. Export CSV → tersimpan ke folder session yang sama
15. Klik tombol Upload to Pinterest → buka Pinterest bulk upload

## FORMAT DESCRIPTION

{2-3 kalimat SEO natural tentang animasi}

Catatan:

## CREDIT LINE FORMAT
- animator: SELALU = uploader_id (akun yang posting di IG)
- client: @mention pemilik logo / "for @", "with @", "made with @"
- logo_maker: @mention logo designer ("logo by @", "design by @")
- Format lengkap: "for @client | Credit: @animator on Instagram | Logo by @logo_maker"
- Tanpa client: "Credit: @animator on Instagram"
- Tanpa logo_maker: "for @client | Credit: @animator on Instagram"

## UPLOADER_ID DETECTION (instagram.py) — Priority order
1. uploader_url field → parse username dari instagram.com/username/
2. uploader_id field → hanya jika bukan numerik dan tidak ada spasi
3. channel field → fallback
4. channel_url → parse username
5. @mention pertama di caption → last resort
